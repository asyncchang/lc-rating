# SegmentTree(線段樹)

## 模板一 查詢 區間最大/小值, 可修改
> 例題 [2398. 預算內的最多機器人數目](https://leetcode.cn/problems/maximum-number-of-robots-within-budget/)
```Python3 []
class SegmentTreeMax:
    def __init__(self, data):
        self.n = len(data)
        self.tree = [0] * (2 * self.n)
        # 建樹
        self._build(data)

    def _build(self, data):
        # 初始化線段樹的葉節點
        for i in range(self.n):
            self.tree[self.n + i] = data[i]
        # 初始化線段樹的內部節點
        for i in range(self.n - 1, 0, -1):
            self.tree[i] = max(self.tree[2 * i], self.tree[2 * i + 1])

    def update(self, index, value):
        # 更新葉節點
        pos = index + self.n
        self.tree[pos] = value
        # 更新線段樹中的相關節點
        while pos > 1:
            pos //= 2
            self.tree[pos] = max(self.tree[2 * pos], self.tree[2 * pos + 1])

    def query(self, left, right): # 查詢區間 [left, right) 的最大值
        left += self.n
        right += self.n
        max_val = -float('inf')
        while left < right:
            if left % 2 == 1:
                max_val = max(max_val, self.tree[left])
                left += 1
            if right % 2 == 1:
                right -= 1
                max_val = max(max_val, self.tree[right])
            left //= 2
            right //= 2
        return max_val
```
