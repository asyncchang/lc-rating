## 模板一
找到每個元素的下一個更大元素（Next Greater Element）
> 例題  [739. 每日溫度](https://leetcode.cn/problems/daily-temperatures/)
```python
def next_greater_element(nums):
    stack = []
    result = [-1] * len(nums)
    for i in range(len(nums)):
        while stack and nums[i] > nums[stack[-1]]:
            result[stack.pop()] = nums[i] # 這裡存的是元素，也可以直接存下標 i
        stack.append(i)
    return result
```

## 模板五
找到每個元素的右側最小的大於等於當前元素的索引
> 例題  [975. 奇偶跳](https://leetcode.cn/problems/odd-even-jump/description/)
```python
def next_min_larger_element(nums):
    n = len(nums)
    a = sorted(range(n), key=lambda i: nums[i]) # 下標按元素值升序
    result = [None] * n
    st = []
    for i in a:
        while st and st[-1] < i: # i 是 st "最小的更大"
            result[st.pop()] = i
        st.append(i)
    return result
```

def next_max_smaller_element(nums):
